# 📱 CallScan AI v4 — Double SIM

> Enregistrez uniquement les appels de la SIM que vous surveillez.
> Les appels de l'autre SIM sont automatiquement ignorés.

## 🚀 Démarrage
```bash
npm install && npm run dev
```

## ⚙️ Configuration initiale
1. Ouvrir l'app → cliquer **Configurer les SIM**
2. Entrer le numéro de la **SIM 1** et de la **SIM 2**
3. Sélectionner quelle SIM surveiller (ex : SIM 1)
4. Ajouter vos contacts en les assignant à SIM 1 ou SIM 2

## 📋 Utilisation
1. Recevoir un appel sur la SIM surveillée
2. Mettre en **haut-parleur**
3. Dans CallScan → appuyer sur **Enregistrer** à côté du contact
4. **Terminer** → sauvegarde automatique dans le bon dossier

## 🔒 Filtrage SIM
- Seuls les contacts assignés à la **SIM active** apparaissent en mode enregistrement
- Les contacts de l'autre SIM sont affichés en grisé avec l'icône 🚫
- Si vous tentez d'enregistrer un contact de la mauvaise SIM → message d'erreur
